1. Se entrena modelo de Yolov8, hay que trabajar en su programación para pulirlo, best.pt es el archivo que contiene el modelo entrenado Yolov8.
2. Hay una carpeta llamada imágenes, contiene 4 imágenes para ensayar su funcionamiento.
3. El archivo llamado requirements son las librerías que necesita el proyecto para funcionar, se puede crear un entorno virtualenv para descargarlas o desde el CMD o Terminal en VScode.
